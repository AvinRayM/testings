Before You Start

1
In Roblox Studio, click ServerScriptService in the Explorer. In the Properties panel, set LoadStringEnabled = true.
2
Go to Game Settings > Security and enable Allow HTTP Requests.
3
Create a Script in ServerScriptService, paste the ServerScript below.
4
Create a LocalScript in StarterPlayerScripts (or inside a GUI TextButton). Paste the LocalScript below.

ServerScript -> ServerScriptService

-- ============================================
-- SERVER SCRIPT (ServerScriptService)
-- Polls the web panel + listens to RemoteEvent
-- ============================================

local HttpService = game:GetService("HttpService")
local ReplicatedStorage = game:GetService("ReplicatedStorage")

----------------------------------------------
-- CHANGE THIS to your Render URL!
-- Example: "https://my-executor.onrender.com"
----------------------------------------------
local API_URL = "YOUR_RENDER_URL"

-- Create RemoteEvent for LocalScript communication
local execRemote = Instance.new("RemoteEvent")
execRemote.Name = "ExecuteRemote"
execRemote.Parent = ReplicatedStorage

-- Admin whitelist (add your UserId)
local ADMINS = {
    [123456789] = true,  -- Replace with YOUR UserId
}

local function executeCode(codeStr)
    local func, compileErr = loadstring(codeStr)
    if not func then
        return false, "Compile Error: " .. tostring(compileErr)
    end
    local success, runErr = pcall(func)
    if not success then
        return false, "Runtime Error: " .. tostring(runErr)
    end
    return true, "Success"
end

-- Listen for in-game RemoteEvent from LocalScript
execRemote.OnServerEvent:Connect(function(player, codeStr)
    if not ADMINS[player.UserId] then
        warn("Blocked execution from non-admin: " .. player.Name)
        return
    end
    print("[Executor] Running code from " .. player.Name)
    local ok, result = executeCode(codeStr)
    if not ok then warn("[Executor] " .. result) end
end)

-- Poll the web backend every 5 seconds
print("[Web Executor] Server started, polling " .. API_URL)
task.spawn(function()
    while task.wait(5) do
        local ok, res = pcall(function()
            return HttpService:GetAsync(API_URL .. "/api/poll")
        end)

        if ok and res then
            local data = HttpService:JSONDecode(res)

            if data and data.jobId and data.script then
                print("[Web Executor] Got job #" .. data.jobId .. ": " .. data.script)

                local execOk, execMsg = executeCode(data.script)

                pcall(function()
                    HttpService:PostAsync(
                        API_URL .. "/api/result",
                        HttpService:JSONEncode({
                            jobId = data.jobId,
                            status = execOk and "Completed" or "Error",
                            error = execMsg
                        }),
                        Enum.HttpContentType.ApplicationJson,
                        false
                    )
                end)
            end
        end
    end
end)

LocalScript -> StarterPlayerScripts or inside a GUI

-- ============================================
-- LOCAL SCRIPT (StarterPlayerScripts or GUI)
-- Fires the RemoteEvent so the Server runs it
-- ============================================

local ReplicatedStorage = game:GetService("ReplicatedStorage")
local execRemote = ReplicatedStorage:WaitForChild("ExecuteRemote")

-- Example: fire a require script to the server
local function runRequire(moduleId, funcName, targetUser)
    local code = "require(" .. tostring(moduleId) .. ")"
    if funcName then
        code = code .. "." .. funcName .. "("
        if targetUser then
            code = code .. '"' .. targetUser .. '"'
        end
        code = code .. ")"
    end
    execRemote:FireServer(code)
    print("Sent to server: " .. code)
end

-- Example usage (runs when the player joins):
runRequire(15827159924, "MauDex", "Builderman")

Deploy to Render (Free)

Render gives you a public URL like https://your-app-name.onrender.com. That URL is what you use as your API URL in the Roblox script AND in your browser to open the web panel.

1

Push your code to GitHub

Your repo needs this file structure:

your-repo/
  backend/
    server.js       <-- the Node server
    package.json    <-- already created for you
  index.html        <-- the web panel

Create a GitHub repo and push these files:

cd your-project-folder
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
2

Create a Render account

Go to https://render.com and sign up with your GitHub account (free).

3

Create a New Web Service

On Render dashboard, click "New +" > "Web Service". Connect your GitHub repo.

Fill in these settings:

Namemy-web-executor
RegionPick the closest to you
Branchmain
Root Directorybackend
RuntimeNode
Build Commandnpm install
Start Commandnode server.js
PlanFree
4

Click "Create Web Service"

Render will build and deploy your app. Wait ~2 minutes. When it says "Live", you're good!

5

Get your URL

Render gives you a URL at the top of the page. It looks like:

https://my-web-executor.onrender.com

This is your API URL! Use it for everything:

  • Open it in your browser = the Web Executor panel
  • Paste it into the Roblox ServerScript where it says YOUR_RENDER_URL
6

Done! Test it

Open your Render URL in the browser. Go to the Execute tab, push a module. Then play your Roblox game — the server script will pick it up in ~5 seconds!

Render Free Tier Note: Free services spin down after 15 minutes of no traffic. The first request after it sleeps takes ~30 seconds to wake up. Your Roblox game constantly polling /api/poll will keep it awake while the game is running. If you want it always on, upgrade to the $7/month plan.
